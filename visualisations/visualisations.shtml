Visualisations

A large part of my current work at University of Oxford is dedicated to data visualisations, as part of the Live Data Project. I'm also generally very interested in the use interactive visualisations as a powerful tool for encouraging open data and open access.

This site provides a useful testing ground for javascript (and other) visualisation technologies where no free-hosting solution exists or is practical. There are therefore a large number of "practice" visualisations on this site, ideally useful visualisations will be functionalised/packaged and provided as distinct repos on my Github account.
