Course description

R is a tremendously popular and powerful scripting language used by millions of analysts and data scientists around the world, and the recently developed tidyverse provides incredible tools for quickly wrangling, cleaning, understanding and visualising data.

This training course covers the following topics:

  • Reproducibly using the tidyverse to import .csv and Excel files into R
  • Programmatically cleaning and wrangling data with dplyr, tidyr and more tidyverse packages
  • Exploratory data analysis using ggplot2
  • Publish publication quality static charts and maps with ggplot2
  • Create rich interactive HTML reports with RMarkdown

  • Design responsive mobile-ready interactive visualisations using the following htmlwidgets packages: leaflet (choropleth), plotly (scatter plots), highcharter (bar charts), visNetwork (network graph)
